# 🤖 Sentiment-Aware Customer Support Chatbot

A simple NLP-based customer support chatbot built using Python and TextBlob. The chatbot can answer common customer queries and provide empathetic responses based on the sentiment of user messages.

## 🚀 Live Demo

**Live App:** [https://sentiment-aware-customer-support-chatbot-eykiltcpzp4nuqnhvo8m5.streamlit.app/]

## 📂 GitHub Repository

**Repository:** https://github.com/AjimTamboli-Web/sentiment-aware-customer-support-chatbot

---

## 📖 Project Overview

This project implements a rule-based customer support chatbot that combines:

* Intent recognition using predefined keywords
* Sentiment analysis using TextBlob
* Customer support automation
* Interactive web interface using Streamlit

The chatbot can answer common questions such as store hours and return policies while also detecting whether a user's message is positive, negative, or neutral.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/AjimTamboli-Web/sentiment-aware-customer-support-chatbot.git
cd sentiment-aware-customer-support-chatbot
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

The application will open in your browser at:

```text
http://localhost:8501
```

## 💬 Sample Interactions

### Store Hours

**User:**

```text
What time do you open?
```

**Bot:**

```text
We are open from 9 AM to 5 PM, Monday to Friday.
```

### Return Request

**User:**

```text
I want a refund
```

**Bot:**

```text
I'd be happy to help you process your return. Let me transfer you to a live agent.
```

### Positive Sentiment

**User:**

```text
I'm really happy today!
```

**Bot:**

```text
That's great to hear!
```

### Negative Sentiment

**User:**

```text
I'm very disappointed.
```

**Bot:**

```text
I'm so sorry to hear that. How can I help?
```
